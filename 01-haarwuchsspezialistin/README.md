# Umstieg zur Haarwuchsspezialistin

Website + Funnel für Ines' Repositionierung weg vom klassischen Friseurgeschäft, hin zur Haarwuchsspezialistin.

Geplanter Funnel: kostenloses PDF → E-Book (siehe `../02-buch-haare-kopfhaut/`) → Fragebogen mit Fotos → persönliche Auswertung (kostenpflichtig) → optional Buchung Beratung online/im Salon.

- `website/` — Seiten-Texte und Struktur (Startseite, Problemseite, Angebote, Über Ines, Ablauf, Kontakt)
- `freebie-pdf/` — kostenloses Einstiegs-PDF
- `ebook/` — E-Book-Fassung als nächster Funnel-Schritt
- `fragebogen-auswertung/` — Fragebogen für Kunden + Vorlage für persönliche Auswertungen

Hintergrund und Regeln (Begriffe, Preise, Stil) stehen im Memory unter `project-haarwuchsspezialistin`.
